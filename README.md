# -FineMed-LLM-Plateforme-d-assistant-m-dical-prudent-LLM-fine-tun-
FineMed‑LLM est une application web qui permet d’interagir avec un LLM médical fine‑tuné afin de générer des réponses plus prudentes et mieux encadrées . Le modèle déployé est Qwen2.5‑1.5B‑Instruct, adapté par QLoRA à partir de cas cliniques simulés et de réponses à risque identifiées lors de l’analyse.
